# Credit Risk Decision System

Projekt hybrydowy łączący analizę systemową z analizą danych. Celem jest zaprojektowanie i symulacja procesu automatycznej oceny zdolności kredytowej klienta.

## Zakres projektu
1. **Analiza Systemowa:**
   - Model procesu biznesowego (BPMN) - ścieżka od wniosku do decyzji.
   - Projekt architektury (UML Sequence) - komunikacja API.
   - Schemat bazy danych (ERD).
2. **Data Science:**
   - Symulator danych (Python) generujący syntetyczne wnioski kredytowe.
   - Analiza ryzyka i wizualizacja wyników.

## Technologie
- **Analiza:** BPMN 2.0, UML, SQL
- **Kod:** Python (Pandas, Random)
- **Status:** W trakcie rozwoju (Work in Progress)
